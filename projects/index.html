Stereo Camera Calibration for YOLO NN Depth Detection/Object Recognition (Python, OpenCV, Linux/Windows)

I took a heavily distorted fish-eye stereo camera and calibrated it with Python OPENCV to return live and static rectified images and videos by calculating each camera's fundamental matrix and epipolar line. After this, I generated live and static depth maps that determined an object's relative distance from the stereo camera system as well as fellow objects around it, for both live video and static images. This in combination with YOLO V4's Object Detection Neural Network allowed me to simultaneously determine an object's identity, as well as its distance from the camera system, all in real time (with the option of static images too!).
